# Gaming Platform Website

A responsive gaming website built with HTML and CSS, featuring user registration, main menu navigation, and detailed game information pages.

## Project Description

This website serves as a front-end interface for a competitive gaming platform. It includes multiple pages for user onboarding, game mode selection, and educational content about ranking systems and gameplay mechanics.

## Features

- **User Registration System**: Clean, accessible form for new account creation
- **Interactive Main Menu**: Visual navigation with game mode selection and vehicle customization options
- **Information Hub**: Comprehensive guide covering game hours, ranking progression, and key features
- **Responsive Design**: Optimized for various screen sizes and devices
- **Accessibility Features**: ARIA labels, semantic HTML, keyboard navigation support
- **Modern UI**: Glassmorphism effects, gradient buttons, and smooth animations
